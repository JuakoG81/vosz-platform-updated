# VOSZ Platform - Servicios y Utilidades

Repositorio con servicios y utilidades del VOSZ Platform.

## Archivos incluidos

- `src/lib/supabase.ts` - Servicios principales de Supabase
- `src/lib/theme.ts` - Sistema de temas y tokens de diseño  
- `src/lib/utils.ts` - Utilidades generales

## Descripción

Este repositorio contiene los servicios y utilidades principales del VOSZ Platform, incluyendo la configuración de Supabase, sistema de temas y funciones utilitarias.
